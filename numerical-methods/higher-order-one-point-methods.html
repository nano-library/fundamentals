Higher order one-point methods: Newton-Raphson method

A one-point method is defined as

\[ x_{n+1} = g(x_n), \quad n \geq 1 \]

where \(g(x_n)\) depends only on the last iterated value.

A General Theory for One-Point Iteration Methods

Consider solving an equation \(x = g(x)\) for a root \(\alpha\) by the iteration:

\[ x_{n+1} = g(x_n), \quad n \geq 1 \]

with an initial guess \(x_0\) to \(\alpha\).

Newton-Raphson method

Suppose a function \(f(x)\) and an initial guess \(x_0\), the Newton-Raphson method approximates the root of the function \(\alpha\) by iterating the following formula:

\[ x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}, \quad n \geq 0 \]

where \(f'(x)\) is the derivative of \(f(x)\) with respect to \(x\).

The key idea behind the method is that a linear approximation of the function \(f(x)\) at the point \(x_n\) can be used to find a better estimate of the root.

The approximation is given by the tangent line at \(x_n\) , which intersects the x-axis at a point that is closer to the root \(\alpha\) than \(x_n\).

Example 1

Find the smallest positive real root of \(\cos(x) - x e^x = 0\), correct to three decimal places.

Solution:

Let \(f(x) = \cos(x) - x e^x\) and \(f'(x) = -\sin(x)-e^x(1+x)\)

Let \(x_0=0.5\)

To obtain the root correct to three decimal places, the error in the approximation must be less than half a unit in the third decimal place.

If a number is correct to three decimal places, then rounding will not change its value provided the error is less than

\[ \frac{1}{2}\times 10^{-3} = 5\times 10^{-4}. \]

Since the exact root is unknown, the absolute error \(|x - \alpha|\) cannot be computed directly. Hence, we use the difference between successive approximations as an estimate of the error. Therefore, we require

\[ |x_{n+1} - x_n| \le 5\times 10^{-4}, \]

which ensures that the approximation is accurate to three decimal places.

Newton-Raphson method approximation formula,

\[ x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}, \quad n \geq 0 \]

Iteration 1

\[ x_1 = x_0 - \frac{f(x_0)}{f'(x_0)} \] \[ f(x_0)=f(0.5)=\cos(0.5)-0.5e^{0.5}\approx 0.0532219 \] \[ f'(x_0)=f'(0.5)=-\sin(0.5)-e^{0.5}(1+0.5)\approx -2.9525074 \] \[ x_1 = 0.5 - \frac{0.0532219}{-2.9525074}\approx 0.5180260 \] \[ |x_1-x_0|=|0.5180260-0.5|\approx 0.0180260 \]

Iteration 2

\[ x_2 = x_1 - \frac{f(x_1)}{f'(x_1)} \] \[ f(x_1)=f(0.5180260)=\cos(0.5180260)-(0.5180260)e^{0.5180260}\approx -8.174369\times 10^{-4} \] \[ f'(x_1)=f'(0.5180260)=-\sin(0.5180260)-e^{0.5180260}(1+0.5180260)\approx -3.0434925 \] \[ x_2 = 0.5180260 - \frac{-8.174369\times 10^{-4}}{-3.0434925}\approx 0.5177574 \] \[ |x_2-x_1|=|0.5177574-0.5180260|\approx 2.6859\times 10^{-4} \]

Since

\[ |x_2-x_1| \approx 2.6859\times 10^{-4} < 5\times 10^{-4}, \]

the approximation is correct to three decimal places. Therefore, the smallest positive real root is

\[ x \approx 0.518. \]
