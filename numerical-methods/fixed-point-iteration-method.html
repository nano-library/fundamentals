Fixed-Point Iteration Method

Definition of Fixed Point

A number \(c\) is a fixed point of a function \(g\) if \(g(c) = c\).

Connection between fixed point and root-finding method

If \(f(c) = 0\), \(g(x) = x-f(x)\) has a fixed point \(c\) as \(g(c) = c\). Conversely, if \(g(c) = c\) , \(f(x) = x - g(x)\) has a root \(c\).

Fixed-Point Iteration Method

Fixed-point method starts with an initial guess \(x_0\) and then generates a sequence using the following formula.

\[ x_{n+1} = g(x_n), \quad n \geq 0 \]

Sufficient conditions for existance and uniqueness of a fixed point

Theorem

Let \(g:[a,b] \rightarrow [a,b]\) be continuous, then \(g\) has a fixed point in \([a,b]\).

If \(g\) is differentiable on \((a,b)\) such that

\[ |g'(x)| \leq L, \quad 0 < L < 1, \forall \; x \in (a,b) \]

then \(g\) has a unique fixed point.

Proof:

Let \(h:[a,b] \rightarrow \mathbb{R}\) be \(h(x) = g(x) - x\).

If

\[ h(c) = 0 \implies g(c) = c \]

\(c\) is a fixed point.

Given

\[ \min_{x \in [a,b]} |g(x)| \geq a \quad \text{and} \quad \max_{x \in [a,b]} |g(x)| \leq b \] \[ a \leq g(x) \leq b \]

Let \(h(a) = g(a) - a \geq 0\) and \(h(b) = g(b) - b \leq 0\)

Case 1:

If

\[ h(a) = 0 \implies g(a) = a \] \[ h(b) = 0 \implies g(b) = b \]

\(a\) and \(b\) are fixed points.

Case 2:

If \(h(a) > 0\) and \(h(b) < 0\), since \(h(x)\) is continuous.

\[ h(c) = 0 \implies g(c) = 0 \]

Uniqueness

Let \(c\) and \(d\) be two fixed points, such that

\[ g(c) = c \; text{and} \; g(d) = d \]

Consider,

\[ |c - d| = |g(c) - g(d)| \]

By Mean Value Theorem,

\[ f(b) - f(a) = f'(c)(b-a) \] \[ |c - d| = |g'(\xi)(c - d)| \]

Since \(g'(\xi) < 1\)

\[ |c - d| < |c - d| \]

which is a contradiction.

Therefore, \(g\) has a unique fixed point.

Picard Fixed Point Interations

Let \(g:[a,b] \rightarrow [a,b]\) be differentiable on \((a,b)\) and \(|g'(x)| \leq L\), \(0 < L < 1\), then for any \(x_0 \in (a,b)\), the sequence

\[ x_{n+1} = g(x_n), \quad n = 0,1,2,3, \ldots \]

converges to a unique fixed point \(c\).

Proof

Consider,

\[ x_n - c = g(x_{n-1}) - g(c) \] \[ x_{n+1} - c = g(x_n) - g(c) = g'(\xi)(x_n - c) \] \[ |x_{n+1} - c| \leq L|x_n - c|, \quad |g'(x)| < L \] \[ |x_{n+1} - c| \leq L \times L|x_n - c| \]

In general,

\[ |x_{n+1} - c| \leq L^{n+1}|x_0 - c| \] \[ \lim_{n \rightarrow \infty} |x_{n+1} - c| = 0, \quad \text{if} \; |L| < 1, \; \text{or} \; 0 < L < 1 \]

Case 1

If \(L = 2\), then

\[ |x_{n+1} - c| \leq 2^{n+1} |x_0 - c| \rightarrow \infty, \quad \text{Diverges} \]

Case 2

If \(L = \frac{1}{2}\), then

\[ |x_{n+1} - c| \leq 2^{-(n+1)} |x_0 - c| \rightarrow 0, \quad \text{Converges} \]

    A fixed point \(c\) of \(g\) satisfies the definition \[ g(c)=c. \] So we solve \[ c^2 - 2 = c. \] Bring all terms to one side: \[ c^2 - c - 2 = 0. \] Factor the quadratic: \[ c^2 - c - 2 = (c-2)(c+1)=0. \] Hence, \[ c = 2 \quad \text{or} \quad c = -1. \]

    Let \(g: [a,b] \rightarrow [a,b]\) be continuous. Then \(x = g(x)\) has at least one solution in \([a,b]\).

    Theorem 2

    Let \(g:[a,b] \rightarrow [a,b]\) be continuous and for some \(\lambda \in (0,1)\)

    \[ |g(x) - g(y)| \leq \lambda|x-y|, \quad \forall \; x,y \in [a,b] \]

    Then \(x = g(x)\) has a unique solution \(r \in [a,b]\).

    Also, the iterations \[x_{n+1} = g(x_n), \quad n \geq 1 \] will converge to \(r\) for any choice of \(x_0 \in [a,b]\) and

    \[ |r - x_n| \leq \frac{\lambda^n}{1- \lambda}|x_1 - x_0| \]

    Theorem 3

    Assume that \(g: [a,b] \rightarrow [a,b]\) is continuously differentiable and

    \[ \lambda = \max_{a \leq x \leq b}|g'(x)| < 1 \]

    Then the following hold:

    1. \(g\) has a unique fixed point \(r \in [a,b]\).

    2. For any \(x_0 \in [a,b]\), the interations \(x_{n+1} = g(x_n)\) converge to r.

    3. The error estimates satisfy \[|r-x_n| \leq \lambda^n|r-x_0| \leq \frac{\lambda^n}{1-\lambda}|x_1 - x_0|\] and \[\lim_{x \rightarrow \infty} \frac{r - x_{n+1}}{r-x_n} = g'(r)\].

    Example

    Let \(r\) be a fixed point of g and suppose \(g\) is continuously differentiable in some neighbourhood of \(r\) with \(|g(r)| < 1\). Then the results of previous theorem still hold provided \(x_0\) is sufficiently close to \(r\).

    Solution:

    Since \(g'\) is continuous at \(r\) and \(g'(r) < 1\), we can pick a number \(lambda\) such that

    \[ |g'(r)| < \lambda < 1 \]

    By continuity of g', there exists \(\delta>0\) such that whenever \(|x - r| < \delta\).

    \[ |g'(x)| \leq \lambda \]

    So on the interval

    \[ I = [r- \delta, r + \delta] \]

    We have

    \[ \max_{x \in I}|g'(x)| \leq \lambda < 1 \]

    Now for any \(x,y \in I\),

    Mean Value Thoerem gives:

    \[ |g(x) - g(y)| = |g'(\xi)||x-y| \leq \lambda |x-y| \]

    So \(g\) is a contradiction on \(I\).

    By Thoerem 3

    If you choose \(x_0 \in I\), then:

    1. \(g\) has a unique fixed point in I (and it must be \(r\)).

    2. The interations \(x_{n+1} = g(x_n)\) stays in \(I\) and converges to r.

    3. The error estimates satisfy \[|r-x_n| \; \text{and} \; \lambda^n|r-x_0| \leq \frac{\lambda^n}{1-\lambda}|x_1 - x_0|\].
