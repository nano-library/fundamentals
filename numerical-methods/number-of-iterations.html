Bisection Method - Number of Iterations

In the bisection method, after \(n\) iterations, the error satisfies

\[ |x_n - c| \le \frac{b-a}{2^n}. \]

To guarantee accuracy within a prescribed tolerance \(\epsilon\), we require

\[ \frac{b-a}{2^n} \le \epsilon. \]

Solving for \(n\):

\[ \frac{b-a}{2^n} = \epsilon \] \[ \log\!\left(\frac{b-a}{2^n}\right) = \log(\epsilon) \] \[ \log(b-a) - n\log(2) = \log(\epsilon) \] \[ n = \frac{\log(b-a) - \log(\epsilon)}{\log(2)}. \]

Since \(n\) must be an integer, we take the smallest integer greater than or equal to this value.

Example 1

Given \[ f(x) = 3x - \cos(x) - 1, \qquad x \in [0,1], \qquad \epsilon = 10^{-3}, \] find the minimum number of iterations required.

Solution

Here, \(a=0\), \(b=1\), so \(b-a = 1\).

\[ n = \frac{\log(1) - \log(10^{-3})}{\log(2)}. \]

Since \(\log(1)=0\) and \(\log(10^{-3}) = -3\) (base 10),

\[ n = \frac{0 - (-3)}{\log(2)} = \frac{3}{0.3010} \approx 9.97. \]

Therefore, the minimum number of iterations required is \(n = 10\).
