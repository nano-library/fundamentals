Root Finding Problem

The process of finding the roots of an equation of the form for a given function \(f\) is known as root-finding or root-solving.

A root of the equation is also called a zero of the function \(f\).

\[ f(x) = 0 \] For this process, we will encounter two types ofnumerical methods.
Direct methods

Methods that find the exact solution to a problem using a finite number of operations.

Iterative method

A mathematical procedure that generates a sequence of approximations to a desired solution by repeatedly applying a fixed rule. It starts with an initial approximation and then generates a new approximation in each iteration, using the result of the previous one.

The iterative method stops when a stopping criterion is met, such as reaching a certain level of accuracy or a maximum number of iterations.
