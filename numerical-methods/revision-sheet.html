Floating Point System (\(\mathbb{F}\))

Using base \(\beta\), any non-zero number \(x\) in a floating-point system is written as

\[ x = s \cdot (.a_1a_2\ldots a_t )_\beta \cdot \beta^{\,e} \] \[ x = (\text{sign})\cdot(\text{fractional part})\cdot(\text{exponent}) \]

Chopping and Rounding

Chopping: Just drop everything after \(t\) digits.

\[ fl(x) = s \cdot (.a_1 \ldots a_t)_\beta \cdot \beta^e \]

Rounding: Add one unit in the last place (ULP) only if the tail is at least half.

\[ fl(x) = s \cdot \Big( (.a_1 \ldots a_t)_\beta + \delta \Big) \cdot \beta^e \] \[ \delta = \begin{cases} (0.00\ldots1)_\beta, & \text{if } a_{t+1} > \frac{\beta}{2} \\ (0.00\ldots1)_\beta, & \text{if tie and further digits } \neq 0 \\ 0, & \text{otherwise} \end{cases} \]

Error In Computer Representation

Suppose that \(x^*\) is an approximation to \(x\).

The absolute error is \[ |x - x^*| \] and the relative error is \[ \frac{|x - x^*|}{|x|} \] provided that \( x \ne 0 \).

General Representation (IEEE 754)

A normalized floating-point number in IEEE 754 format is represented as

\[ (-1)^{\text{sign}} \times 1.\text{mantissa} \times 2^{\text{exponent} - \text{bias}} \]
  • Sign bit: Determines whether the number is positive or negative.
  • Mantissa (Fraction): Stores the significant digits. In normalized form, the leading 1 is implicit.
  • Exponent: Stored with a bias to allow representation of both positive and negative exponents.

IEEE Single Precision (32-bit)

  • 32-bit word
  • 1 bit for sign
  • 8 bits for biased exponent
  • 23 bits for fractional part (mantissa)

IEEE Double Precision (64-bit)

  • 64-bit word (or two 32-bit words)
  • 1 bit for sign
  • 11 bits for biased exponent
  • 52 bits for fractional part (mantissa)

Conversion to IEEE 754 Format

  • 1. Sign: \[ s = \begin{cases} 0 & \text{if } x \ge 0 \\ 1 & \text{if } x < 0 \end{cases} \]
  • 2. Convert to binary: Write \(x\) in base 2.
  • 3. Normalize: \[ x = 1.a_1a_2a_3\ldots \times 2^n \]
  • 4. Biased exponent: \[ E = n + \text{Bias} \] (127 single, 1023 double)
  • 5. Mantissa: Store fractional part \(a_1a_2\ldots\)
    23 bits (single), 52 bits (double)

Final form:

\[ (-1)^s \times 1.\text{mantissa} \times 2^{(E - \text{Bias})} \]

Decimal to IEEE 754 (Single Precision Example)

Example: Convert \(x = 13.25\)

  • 1. Sign: \(x>0 \Rightarrow s=0\)
  • 2. Convert to binary:
    \(13_{10}=1101_2\)
    \(0.25_{10}=0.01_2\)
    \[ 13.25_{10}=1101.01_2 \]
  • 3. Normalize: \[ 1101.01_2 = 1.10101_2 \times 2^3 \] so \(n=3\)
  • 4. Biased exponent: \[ E = n + 127 = 3 + 127 = 130 \] \[ 130_{10}=10000010_2 \]
  • 5. Mantissa: Drop leading 1 → \(10101\)
    Pad to 23 bits: \[ 10101000000000000000000 \]

Final IEEE 754 (32-bit):

\[ \boxed{ 0\;|\;10000010\;|\;10101000000000000000000 } \]

Root Finding Methods

Bisection Method

  • Condition: \[ f(a) \cdot f(b) < 0 \] \(\implies\) there exists root in \([a,b]\)
  • Iteration: \[ x_n = \frac{a_n + b_n}{2} \] Keep subinterval where sign changes.
  • Error bound: \[ |x_n - c| \le \frac{b-a}{2^n} \]
  • Iterations for tolerance \(\epsilon\): \[ n \ge \frac{\log(b-a) - \log(\epsilon)}{\log 2} \]
  • Convergence: \[ e_{n+1} \approx \frac{1}{2}e_n \] Order \(p=1\), \(M=\frac{1}{2}\)

Secant Method

  • Iteration: \[ x_{n+1} = x_n - f(x_n)\frac{x_n-x_{n-1}}{f(x_n)-f(x_{n-1})} \]
  • Error relation: \[ e_{n+1} \approx M_1 e_n e_{n-1}, \qquad M_1=\frac{f''(c)}{2f'(c)} \]
  • Order of convergence: \[ p = 1+\frac{1}{p} \Rightarrow p=\frac{1+\sqrt5}{2}\approx1.618 \]
  • Asymptotic form: \[ e_{n+1}\approx C e_n^{\,p}, \qquad C=|M_1|^{1/p} \]
  • Final error model: \[ |x_{n+1}-c| \approx \left| \frac{f''(c)}{2f'(c)} \right|^{\frac{\sqrt5-1}{2}} |x_n-c|^{\frac{\sqrt5+1}{2}} \]
  • Convergence: Superlinear, order \(p\approx1.618\).

Regula-Falsi (False Position)

  • Condition (bracketing): \[ f(a) \cdot f(b) < 0 \] \(\implies\) there exist a root in \([a,b]\)
  • Formula (same as Secant on endpoints): \[ x_{n+1} = x_n - f(x_n) \frac{x_n-x_{n-1}}{f(x_n)-f(x_{n-1})} \]
  • Update rule (keep root bracketed):
    If \(f(x_{n+1})f(x_{n-1}) < 0\) → new interval \([x_{n-1},x_{n+1}]\)
    If \(f(x_{n+1})f(x_n) < 0\) → new interval \([x_{n+1},x_n]\)
  • Idea: Secant step + Bisection safety.
  • Convergence: Linear (usually faster than Bisection).

Newton-Raphson Method

  • One-point form: \[ x_{n+1} = g(x_n) \]
  • Newton iteration: \[ x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)} \]
  • Idea: Use tangent at \(x_n\) to approximate root.
  • Error relation (near root \(\alpha\)): \[ e_{n+1} \approx \frac{f''(\alpha)}{2f'(\alpha)}\, e_n^2 \]
  • Convergence: Quadratic, order \(p=2\).

Fixed-Point Iteration

  • Fixed point: \[ g(c)=c \]
  • Iteration: \[ x_{n+1}=g(x_n) \]
  • Existence: If \(g:[a,b]\to[a,b]\) continuous ⇒ at least one fixed point.
  • Uniqueness + Convergence: If \[ |g'(x)|\le L< 1 \] then unique fixed point \(r\) and \[ |x_{n+1}-r|\le L|x_n-r| \]
  • Error bound: \[ |x_n-r|\le L^n|x_0-r| \]
  • Convergence condition:Contractive map \(\implies\) linear convergence.

Modified Newton's Method

  • Multiple root: \[ f(x)=(x-r)^m q(x), \quad q(r)\neq0 \]
  • Standard Newton: \[ x_{n+1}=x_n-\frac{f(x_n)}{f'(x_n)} \] For multiplicity \(m>1\): \[ g'(r)=1-\frac{1}{m}\neq0 \] ⇒ only linear convergence.
  • Modified Newton: \[ x_{n+1} = x_n - m\frac{f(x_n)}{f'(x_n)} \]
  • Result: Restores quadratic convergence for root of multiplicity \(m\).
  • Multiplicity test: \[ f(r)=f'(r)=\dots=f^{(m-1)}(r)=0, \quad f^{(m)}(r)\neq0 \]

Newton's Method for Systems

  • System form: \[ f(x)=0, \quad x= \begin{bmatrix} x_1\\ \vdots\\ x_n \end{bmatrix}, \quad f(x)= \begin{bmatrix} f_1\\ \vdots\\ f_n \end{bmatrix} \]
  • Jacobian: \[ J(x)=\left[\frac{\partial f_i}{\partial x_j}\right] \]
  • Newton iteration: \[ x^{k+1} = x^k - J(x^k)^{-1}f(x^k) \]
  • Practical form (no inverse): Solve \[ J(x^k)h=-f(x^k) \] then update \[ x^{k+1}=x^k+h \]
  • \(2 \times 2\) case: \[ \begin{bmatrix} f_x & f_y\\ g_x & g_y \end{bmatrix} \begin{bmatrix} h\\k \end{bmatrix} = \begin{bmatrix} -f\\-g \end{bmatrix} \]
  • Condition: \[ \det(J)\neq0 \]
  • Convergence: Quadratic (if initial guess is close and \(J\) nonsingular).

Gaussian Elimination

  • Goal: \[ AX=b \;\longrightarrow\; Ux=b' \] where \(U\) is upper triangular.
  • Forward Elimination (k = 1 to n-1):
    Multipliers: \[ m_{ik}=\frac{a_{ik}^{(k)}}{a_{kk}^{(k)}}, \quad i=k+1,\dots,n \] Row update: \[ a_{ij}^{(k+1)}=a_{ij}^{(k)}-m_{ik}a_{kj}^{(k)} \] \[ b_i^{(k+1)}=b_i^{(k)}-m_{ik}b_k^{(k)} \]
  • Condition: \[ a_{kk}^{(k)} \neq 0 \]
  • Back Substitution: \[ x_n=\frac{b_n}{a_{nn}} \] \[ x_i= \frac{b_i-\sum_{j=i+1}^{n}a_{ij}x_j}{a_{ii}}, \quad i=n-1,\dots,1 \]
  • Idea: Eliminate below diagonal → solve upward.

Operation Count - Gaussian Elimination (Total)

  • Multiplications / Divisions: \[ \frac{n^3}{3} + n^2 - \frac{n}{3} \]
  • Additions / Subtractions: \[ \frac{n^3}{3} + \frac{n^2}{2} - \frac{5n}{6} \]
  • Leading term (dominant cost): \[ \frac{2}{3}n^3 + O(n^2) \]
  • Meaning of terms:
    \( \frac{2}{3}n^3 \) → main computational cost (forward elimination)
    \( O(n^2) \) → lower-order cost (back substitution + smaller terms)

Pivoting

  • Pivot problem: small/zero pivot \(a_{kk}^{(k)}\) makes \[ m_{ik}=\frac{a_{ik}^{(k)}}{a_{kk}^{(k)}} \] unstable/undefined.
  • Partial pivoting (row swaps only): choose \(p\) in column \(k\) \[ |a_{p,k}^{(k)}|=\max_{i=k,\dots,n}|a_{i,k}^{(k)}| \] then swap rows \(k \leftrightarrow p\).
  • Complete pivoting (row + column swaps): choose \((p,q)\) in remaining submatrix \[ a_{p,q}^{(k)}|=\max_{i,j=k,\dots,n}|a_{i,j}^{(k)}| \] then swap rows \(k \leftrightarrow p\), columns \(k \leftrightarrow q\) (also swap variables).

Doolittle LU

  • Factorization: \[ A=LU \] \(L\): lower triangular with diagonal \(1\)'s (unit lower), \(U\): upper triangular.
  • Existence (no pivoting): \[ D_k\neq0,\quad k=1,\dots,n \] (all leading principal minors nonzero).
  • Entry formulas: \[ u_{ij}=a_{ij}-\sum_{k=1}^{i-1} \ell_{ik}u_{kj},\quad i\le j \] \[ \ell_{ij}=\frac{a_{ij}-\sum_{k=1}^{j-1}\ell_{ik}u_{kj}}{u_{jj}},\quad i>j \] with \(\ell_{ii}=1\).
  • Uniqueness: if \(A=L_1U_1=L_2U_2\) with unit-lower \(L\), then \[ L_1=L_2,\quad U_1=U_2 \]
  • Solve \(Ax=b\): \[ Ly=b \quad(\text{forward subst.}),\qquad Ux=y \quad(\text{back subst.}) \]

Crout's Decomposition

  • Factorization: \[ A=LU \] \(L\): lower triangular (diagonal not forced to 1), \(U\): unit upper triangular (\(u_{ii}=1\)).
  • Entry formulas: \[ \ell_{ij}=a_{ij}-\sum_{k=1}^{j-1}\ell_{ik}u_{kj},\quad i\ge j \] \[ u_{ij}=\frac{a_{ij}-\sum_{k=1}^{i-1}\ell_{ik}u_{kj}}{\ell_{ii}},\quad i< j \] with \(u_{ii}=1\).
  • Existence condition (no pivoting): \[ \ell_{ii}\neq0,\quad i=1,\dots,n \]
  • Solve \(Ax=b\): \[ Ly=b \quad(\text{forward}),\qquad Ux=y \quad(\text{back}) \]
  • Doolittle vs Crout: Doolittle: \(\ell_{ii}=1\). Crout: \(u_{ii}=1\).
