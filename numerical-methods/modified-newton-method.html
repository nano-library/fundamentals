Modified Newton's Method

A solution \(r\) of \(f(x) = 0\) is a zero of multiplicity \(m\) of \(f\) if for \(x \neq r\), we can write

\[ f(x) = (x-r)^m q(x) \]

for some continuous function \(q(x)\) such that \(\lim_{x \rightarrow r}q(x) \neq 0\).

Suppose \(r\) is a zero of \(f\) of multiplicity \(m\) with \(f(x) = (x - r)^m q(x)\).

From the Newton-Raphson method,

\[ x_{n+1} = g(x_n) \]

with

\[ g(x) = x - \frac{f(x)}{f'(x)} \]

Now, We can write

\[ g(x) = x-\frac{(x-r)^m q(x)}{m(x-r)^{m-1}q(x)+(x-r)^m q'(x)} \] \[ g(x) = x-\frac{(x-r)q(x)}{m q(x) + (x-r)q'(x)} \]

Then

\[ g'(x) = 1-\frac{q(x)}{mq(x)+(x-r)q'(x)} - (x-r)\frac{d}{dx}\left( \frac{q(x)}{mq(x)+(x-r)q'(x)} \right) \]

This implies \(g'(r) = 1-\frac{1}{m} \neq 0\).

Therefore, in this case we only have linear rate of convergence.

To recover the quadratic rate of convergence one can use the following modified Newton's method.

\[ x_{n+1} = x_n - m\frac{f(x_n)}{f'(x_n)}, \quad n \geq 0 \]

where \(m\) is the multiplicity of the root of \(f\).

Note: If \(f(\alpha) = f'(\alpha) = ... = f^{(m-1)}(\alpha) = 0\) and \(f^{(m)}(\alpha) \neq 0,\), then \(\alpha\) is a root of \(f\) with multiplicity \(m\).

Example 1

Show that \(f(x) = e^{x^2}-1\) has a root \(0\) with multiplicity \(2\). Then apply modified Newton's method to find the root.

Solution:

If \(f(\alpha) = f'(\alpha) = ... = f^{(m-1)}(\alpha) = 0\) and \(f^{(m)}(\alpha) \neq 0,\), then \(\alpha\) is a root of \(f\) with multiplicity \(m\).

Here,

\[ f(0) = 0 \] \[ f'(x) = 2xe^{x^2}, f'(0) = 0 \] \[ f''(x) = 2e^{x^2} + 4x^2e^{x^2}, f''(0) = 2 \neq 0 \]

Hence \(f\) has root \(0\) with multiplicity \(2\)

\[ \begin{aligned} x_{n+1} &= x_n - 2 \frac{f(x_n)}{f'(x_n)}, \qquad n \geq 0 \\ &= x_n - 2 \frac{e^{x_n^2} - 1}{2x_n e^{x_n^2}} \\ &= x_n - \frac{e^{x_n^2} - 1}{x_n e^{x_n^2}}. \end{aligned} \]

Time to choose one initial guess. Let us take \(x_0 = 0.5\). Note that in Newton's method we cannot choose \(x_0\) such that \(f'(x_0)=0\).

Iteration 1

\[ x_1 = x_0 - \frac{e^{x_0^2} - 1}{x_0 e^{x_0^2}} \] \[ x_1 = 0.5 - \frac{e^{0.5^2} - 1}{0.5 \cdot e^{0.5^2}} \] \[ x_1 = 0.0576 \]

Iteration 2

\[ x_2 = x_1 - \frac{e^{x_1^2} - 1}{x_1 e^{x_1^2}} \] \[ x_2 = 0.0576 - \frac{e^{0.0576^2} - 1}{0.0576 \cdot e^{0.0576^2}} \] \[ x_2 = 9.5446 \times 10^{-5} \]

Iteration 3

\[ x_3 = x_2 - \frac{e^{x_2^2} - 1}{x_2 e^{x_2^2}} \] \[ x_2 = (9.5446 \times 10^{-5}) - \frac{e^{(9.5446 \times 10^{-5})^2} - 1}{(9.5446 \times 10^{-5}) \cdot e^{(9.5446 \times 10^{-5})^2}} \] \[ x_3 = 5.8160 \times 10^{-12} \]

which is almost zero.
