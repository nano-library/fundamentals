Modified Newton's Method

A solution \(r\) of \(f(x) = 0\) is a zero of multiplicity \(m\) of \(f\) if for \(x \neq r\), we can write

\[ f(x) = (x-r)^m q(x) \]

for some continuous function \(q(x)\) such that \(\lim_{x \rightarrow r}q(x) \neq 0\).

Suppose \(r\) is a zero of \(f\) of multiplicity \(m\) with \(f(x) = (x - r)^m q(x)\).

From the Newton-Raphson method,

\[ x_{n+1} = g(x_n) \]

with

\[ g(x) = x - \frac{f(x)}{f'(x)} \]

Now, We can write

\[ g(x) = x-\frac{(x-r)^m q(x)}{m(x-r)^{m-1}q(x)+(x-r)^m q'(x)} \] \[ g(x) = x-\frac{(x-r)q(x)}{m q(x) + (x-r)q'(x)} \]

Then

\[ g'(x) = 1-\frac{q(x)}{mq(x)+(x-r)q'(x)} - (x-r)\frac{d}{dx}\left( \frac{q(x)}{mq(x)+(x-r)q'(x)} \right) \]

This implies \(g'(r) = 1-\frac{1}{m} \neq 0\).

Therefore, in this case we only have rate of convergence.

To recover the quadratic rate of convergence one can use the following modified Newton's method.

\[ x_{n+1} = x_n - m\frac{x_n}{f'(x_n)}, \quad n \geq 0 \]

where \(m\) is the multiplicity of the root of \(f\).

Note

If

\[ f(\alpha) = f'(\alpha) = ... = f^{(m-1)}(\alpha) = 0 \]

and

\[ f^{(m)}(\alpha) \neq 0, \]

then \(\alpha\) is a root of \(f\) with multiplicity \(m\).

Example

Show that \(f(x) = e^{x^2}-1\) has a root \(0\) with multiplicity \(2\). Then apply modified Newton's method to find the root.

Solution:

If

\[ f(\alpha) = f'(\alpha) = ... = f^{(m-1)}(\alpha) = 0 \]

and

\[ f^{(m)}(\alpha) \neq 0, \]

then \(\alpha\) is a root of \(f\) with multiplicity \(m\).

Here,

\[ f(0) = 0 \] \[ f'(x) = 2xe^{x^2}, f'(0) = 0 \] \[ f''(x) = 2e^{x^2} + 4x^2e^{x^2}, f''(0) = 2 \neq 0 \]

Hence \(f\) has root \(0\) with multiplicity \(2\)

\[ \begin{aligned} x_{n+1} &= x_n - 2 \frac{f(x_n)}{f'(x_n)}, \qquad n = 0,1,2,\ldots \\ &= x_n - 2 \frac{e^{x_n^2} - 1}{2x_n e^{x_n^2}} \\ &= x_n - \frac{e^{x_n^2} - 1}{x_n e^{x_n^2}}. \end{aligned} \]

Time to choose one initial guess. Let us take \(x_0 = 0.5\). Note that in Newtonâ€™s method we cannot choose \(x_0\) such that \(f'(x_0)=0\). Then use a calculator to calculate the following:

\(x_0 = 0.5\)
\(x_1 = 0.0576\)
\(x_2 = 9.5446 \times 10^{-5}\)
\(x_3 = 5.8160 \times 10^{-12}\)

which is almost zero.

Example 1

Use modified Newton's method to find solutions accurate to within \(10^{-5}\) to \[ 1 - 4x\cos x + 2x^2 + \cos 2x = 0, \qquad 0 \le x \le 1. \]

Solution:

Step 1: Define \(f(x)\) and simplify (to see the multiplicity)

Let \[ f(x)=1 - 4x\cos x + 2x^2 + \cos 2x. \] Use the identity \(\cos 2x = 2\cos^2 x - 1\): \[ f(x)=1 - 4x\cos x + 2x^2 + (2\cos^2 x - 1) =2\cos^2 x - 4x\cos x + 2x^2 =2(\cos x - x)^2. \]

So \(f(x)=0 \iff (\cos x - x)^2=0 \iff \cos x = x\). Since \(f(x)\) is a square, the root in \([0,1]\) has multiplicity \(m=2\).

Step 2: Compute \(f'(x)\)

Differentiate the original form: \[ f'(x)=\frac{d}{dx}\Big(1 - 4x\cos x + 2x^2 + \cos 2x\Big) = (-4\cos x + 4x\sin x) + 4x - 2\sin 2x. \] So \[ f'(x)= -4\cos x + 4x\sin x + 4x - 2\sin 2x. \]

Step 3: Modified Newton iteration (for multiplicity \(m=2\))

The modified Newton method is \[ x_{n+1}=x_n - m\frac{f(x_n)}{f'(x_n)}. \] (Notice: it is \(m\frac{f(x_n)}{f'(x_n)}\), not \(m\frac{x_n}{f'(x_n)}\).) For this problem \(m=2\), hence \[ x_{n+1}=x_n - 2\frac{f(x_n)}{f'(x_n)}. \]

Step 4: Iterations (take \(x_0=0.5\))

We iterate until \(|x_{n+1}-x_n|<10^{-5}\).

\(n\) \(x_n\) \(|x_{n}-x_{n-1}|\)
0 \(0.5000000000\) --
1 \(0.7552224171\) \(2.552224171\times 10^{-1}\)
2 \(0.7391416661\) \(1.608075096\times 10^{-2}\)
3 \(0.7390851339\) \(5.653223000\times 10^{-5}\)
4 \(0.7390851620\) \(2.812261700\times 10^{-8}\)

Since \(|x_4-x_3| \approx 2.8\times 10^{-8} < 10^{-5}\), we stop. Therefore, the solution in \([0,1]\) accurate to within \(10^{-5}\) is \[ \boxed{x \approx 0.73909}. \]
