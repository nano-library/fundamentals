Bisection Method

The first technique, based on the Intermediate Value Theorem, is called the Bisection, or Binary-search, method.

Suppose \(f\) is a continuous function defined on the interval \([a, b]\), with \(f(a)\) and \(f(b)\) of opposite sign. By the Intermediate Value Theorem, it follows that \(f(x)\) has at least one root in the interval \([a, b]\).

\[ f(a) \cdot f(b) < 0 \]

The method calls for a repeated halving (or bisecting) of subintervals of \([a, b]\) and, at each step, locating the half containing \(x\)

To initiate the method, we set \(a_1 = a\) and \(b_1 = b\). Let \(x_0\) be midpoint in the interval \([a, b]\).

The first approximation of the root is given by

\[ x_1 = \frac{a_1 + b_1}{2} \]

If \(f(x) = 0\), then \(x=x_1\), \(x_1\) is the root.

If \(f(x_1 \neq 0)\) and \(f(a_1)\) have same sign, \(x \in (x_1, b_1)\). Set \(a_2 = x_1\) and \(b_2 = b_1\).

If \(f(x_1 \neq 0)\) and \(f(a_1)\) have opposite signs, \(x \in (a_1, x_1)\). Set \(a_2 = a_1\) and \(b_2 = x_1\).

The Bisection method, though conceptually clear, has significant drawbacks. It is relatively slow to converge (that is, N may become quite large before \(|p-p_N\) is sufficiently small), and a good intermediate approximation might be inadvertently discarded.

Each successive pair of approximations in the Bisection method brackets a root \(x\) of the equation; that is, for each positive integer \(n\), a root lies between \(a_n\) and \(b_n\). This implies that, for each \(n\), the Bisection method iterations satisfy

\[ |x_n - x| < \frac{1}{2}|a_n - b_n| \]

which provides an easily calculated error bound for the approximations.

    • Example 1

    Show that \(f(x) = x^3 + 4x^2 - 10 = 0\) has a root in \([1,2]\), and use the Bisection method to determine an approximation to the root that is accurate to at least within \(10^{-4}\).

    Solution

    Substituting interval endpoints

    \[ f(1) = (1)^3 + 4(1)^2 - 10 = -5 \; \text{and} \; f(2) = 2^3 + 4(2)^2 - 10 = 14 \]

    By Intermediate Value Theorem,

    \[ f(1) \cdot f(2) = -70 < 0 \]

    Hence, root lies in \([1,2]\).

    For bisection, after \(n\) iterations, the interval length is

    \[ \frac{b_1 - a_1}{2^n} \]

    and the midpoint \(x_n\) satisfies the absolute error bound

    \[ |x - x_n| \leq \frac{b_1 - a_1}{2^{n+1}} \]

    Here,

    \[ \frac{2-1}{2^{n+1}} = \frac{1}{2^{n+1}} \leq 10^{-4} \] \[ 2^{n+1} \geq 10^{-4} \] \[ n+1 \geq 14 \] \[ n \geq 13 \]

    We need at least 13 iterations.

    For the first iteration of the Bisection method we use the fact that at the midpoint of \([1,2]\).

    \[ x_1 = \frac{1+2}{2} = 1.5 \] \[ f(x_1) = (1.5)^3 + 4(1.5)^2 - 10 = 2.375 > 0 \]

    Now,

    If \(f(x_1) \cdot f(a_1) > 0\). Set \(a_2 = x_1\) and \(b_2 = b_1\).

    If \(f(x_1) \cdot f(a_1) < 0\). Set \(a_2 = a_1\) and \(b_2 = x_1\).

    Here,

    \[ f(x_1) \cdot f(1) = -11.875 < 0 \]

    This indicates that we should select the interval \([1, 1.5]\) for our second iteration.

    For the second iteration of the Bisection method we use the fact that at the midpoint of \([1,1.5]\).

    \[ x_2 = \frac{1+1.5}{2} = 1.25 \] \[ f(1.25) = (1.25)^3 + 4(1.25)^2 - 10 = -1.796875 < 0 \]

    Now,

    If \(f(x_2) \cdot f(a_2) > 0\). Set \(a_3 = x_2\) and \(b_3 = b_2\).

    If \(f(x_2) \cdot f(a_2) < 0\). Set \(a_3 = a_2\) and \(b_3 = x_2\).

    Here,

    \[ f(x_2) \cdot f(1) = 3.26 > 0 \]

    This indicates that we should select the interval \([1.25, 1.5]\) for our second iteration.

    After 13 iterations, \(x_{13} = 1.365112305\) approximates the root \(x\) with an error.

    Error:

    Note that \(x_9\) is closer to \(x\) than is the final approximation \(x_{13}\). You might suspect this is true because \(|f(x_9)| < |f(x_{13})|\), but we cannot be sure of this unless the true answer is known.

    Example 2

    Determine the number of iterations necessary to solve \(f(x) = x^3 + 4x^2 - 10 = 0\) with accuracy \(10^{-3}\) using \(a_1 = 1\) and \(b_1 = 2\).

    Solution

    We we will use logarithms to find an integer '\(n\)' that satisfies:

    \[ |x_n - x| \leq 2^{-n}(b-a) \]

    We have

    \[ |x_n - x| \leq 2^{-n}(2-1) \] \[ |x_n - x| \leq 2^{-n} < 10^{-3} \]

    Logarithms to any base would suffice, but we will use base-10 logarithms because the tolerance is given as a power of 10.

    \[ \log_{10}{2^{-n}} \leq \log_{10}{10^{-3}} \] \[ -n\log_{10}2 < -3\log_{10}{10} \]

    Since \(\log_{10}{10} = 1\),

    \[ -n\log_{10}2 < -3 \] \[ n > \frac{3}{\log_{10}{2}} \approx 9.96 \]

    Hence, 10 iterations will ensure an approximation accurate to within \(10^{-3}\)

    In Example 1, table shows that the value of \(x_9 = 1.365234375\) is accurate to within \(10^{-4}\). Again, it is important to keep in mind that the error analysis gives only a bound for the number of iterations. In many cases this bound is much larger than the actual number required.

    Example 3

    Find the smallest positive real roots of \(\cos{x}-xe^x = 0\), correct to three decimal places.

    Solution

    Let \(a_0=0.5,\; b_0=0.6\)

    Substituting interval endpoints

    \[ f(0.5)=\cos(0.5)-0.5e^{0.5}\approx 0.05322 \] \[ f(0.6)=\cos(0.6)-0.6e^{0.6}\approx -0.26794 \]

    By Intermediate Value Theorem,

    \[ f(0.5) \cdot f(0.6) < 0 \]

    Hence, smallest positive root lies in \([0.5,\,0.6]\).

    For bisection, after \(n\) iterations, the interval length is

    \[ \frac{b_1 - a_1}{2^n} \]

    and the midpoint \(x_n\) satisfies the absolute error bound

    \[ |x - x_n| \leq \frac{b_1 - a_1}{2^{n+1}} \]

    Here,

    \[ \frac{0.6-0.5}{2^{n+1}} = \frac{0.1}{2^{n+1}} \] \[ \frac{0.1}{2^{n+1}} \leq 10^{-3} \] \[ 2^{n+1} \geq 10^2 \]

    We will use base-10 logarithms because the tolerance is given as a power of 10.

    \[ \log_{10}{2^{n+1}} \geq \log_{10}{10^2} \] \[ (n+1)\log_{10}{2} \geq 2\log_{10}{10} \]

    Since \(\log_{10}{10} = 1\),

    \[ (n+1)\log_{10}{2} \geq 2 \] \[ n+1 \geq \frac{2}{\log_{10}{2}} \] \[ n+1 \geq 6.64 \] \[ n \geq 5.64 \]

    We need at least 6 iterations.

    For the first iteration of the Bisection method we use the fact that at the midpoint of \([0.5,0.6]\).

    \[ x_1 = \frac{0.5+0.6}{2} = 0.55 \] \[ f(x_1) = \cos{0.55} - (0.55)e^{0.55} = -0.100 < 0 \]

    Now,

    If \(f(x_1) \cdot f(a_1) > 0\). Set \(a_2 = x_1\) and \(b_2 = b_1\).

    If \(f(x_1) \cdot f(a_1) < 0\). Set \(a_2 = a_1\) and \(b_2 = x_1\).

    Here,

    \[ f(x_1) \cdot f(0.5) = -0.00532 < 0 \]

    This indicates that we should select the interval \([0.5, 0.55]\) for our second iteration.

    For the second iteration of the Bisection method we use the fact that at the midpoint of \([0.5,0.55]\).

    \[ x_2 = \frac{0.55+0.6}{2} = 1.25 \] \[ f(1.25) = (1.25)^3 + 4(1.25)^2 - 10 = -1.796875 < 0 \]

    Now,

    If \(f(x_2) \cdot f(a_1) > 0\). Set \(a_3 = x_2\) and \(b_3 = b_2\).

    If \(f(x_2) \cdot f(a_1) < 0\). Set \(a_3 = a_2\) and \(b_3 = x_2\).

    Here,

    \[ f(x_2) \cdot f(0.5) = -0.095 < 0 \]

    This indicates that we should select the interval \([0.5, 0.525]\) for our second iteration.

    Bisection iterations \[ \begin{array}{c|c|c|c|c|c} n & a & b & c=\frac{a+b}{2} & f(c) & \text{Next interval} \\ \hline 1 & 0.5000 & 0.6000 & 0.5500 & -0.1008 & [0.5000,\,0.5500] \\ 2 & 0.5000 & 0.5500 & 0.5250 & -0.0222 & [0.5000,\,0.5250] \\ 3 & 0.5000 & 0.5250 & 0.5125 & +0.0159 & [0.5125,\,0.5250] \\ 4 & 0.5125 & 0.5250 & 0.5188 & -0.0030 & [0.5125,\,0.5188] \\ 5 & 0.5125 & 0.5187 & 0.5156 & +0.0065 & [0.5156,\,0.5188] \\ 6 & 0.5156 & 0.5187 & 0.5172 & +0.0017 & [0.5172,\,0.5188] \\ 7 & 0.5172 & 0.5187 & 0.5180 & -0.0006 & [0.5172,\,0.5180] \\ \end{array} \]

    After 7 steps, the interval is:

    \[ [\,0.5172,\;0.5187\,] \]

    Its error is \(0.5179-0.5187=0.0008<0.001\), so the root is determined to three decimal places.

    The smallest positive real root is: \(x \approx 0.518\).
