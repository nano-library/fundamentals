Order of Convergence

Order of Convergence Theorem

Let

\[ e_n = x_n - c \] \[ e_{n+!} = x_{n+1} - c \]

where \(c\) is the exact root.

If

\[ \lim_{n \rightarrow \infty}\frac{|x_{n+1} - c|}{|x_n - c|^p} = M, \quad M \neq 0, p > 0 \]

then order of iterative scheme is \(p\)

\(M\) is called asymptotic error constant.

\[ e_{n+1} \simeq Me_n^p \]

Example 1:

For \(x_{n+1} = x_n^2\), with \(x_0 = \frac{1}{3}\)

\[ c=0 \] \[ |x_{n+1} - c| = |x_n - c|^2 \] \[ \frac{|x_{n+1} - c|}{|x_n - c|^2} = M \] \[ e_{n+1} = e_n^2 \]

Order of Convergence is \(2\).

Asymptotic error is \(1\).

Note

\[ e_{n+1} = c_1e_n + c_2e_n^2 + c_3e_n^3 + \ldots + c_n\frac{e^n_n}{n!} + \ldots \]

Example:

For Order = 3

\[ c_1 = 0, \; c_2 = 0, c_3 \neq 0 \]

Bisection Method

Let \(x_n\) be a sequence of real numbers and \(c\) is a root of \(f(x) = 0, \; x \in [a,b]\). Then

\[ |x_n - c| \leq \frac{b-a}{2^n}, \] \[ |x_{n+!} - c| \leq \frac{b-a}{2^{n+1}}, \] \[ e_n \simeq \frac{b-a}{2^n} \] \[ e_{n+1} \simeq \frac{b-a}{2^{n+1}} \]

Consider,

\[ \frac{e_{n+1}}{e_n} = \frac{1}{2} \implies e_{n+1} \simeq \frac{1}{2}e_n \]

Order, \(p=1\)

Asymptotic error constant, \(M = \frac{1}{2}\)

Secant Method

Let \(f(c)=0\) and assume \(f'(c)\neq 0\) and \(f\) is sufficiently smooth near \(c\). Define the errors \[ e_n = x_n - c,\qquad e_{n-1}=x_{n-1}-c,\qquad e_{n+1}=x_{n+1}-c. \]

The Secant iteration is \[ x_{n+1}=x_n - f(x_n)\,\frac{x_n-x_{n-1}}{f(x_n)-f(x_{n-1})},\qquad n\ge 1. \] Substituting \(x_n=c+e_n\) and \(x_{n-1}=c+e_{n-1}\) gives \[ \begin{aligned} c+e_{n+1} &=(c+e_n)-f(c+e_n)\, \frac{(c+e_n)-(c+e_{n-1})}{f(c+e_n)-f(c+e_{n-1})} \\ &= (c+e_n)-f(c+e_n)\,\frac{e_n-e_{n-1}}{f(c+e_n)-f(c+e_{n-1})}. \end{aligned} \] (Note: the numerator must involve \(e_{n-1}\), not \(e_{n+1}\), because the method uses \(x_n\) and \(x_{n-1}\).)

Use Taylor expansion about \(c\) (with \(f(c)=0\)): \[ f(c+e_n)= e_n f'(c)+\frac{e_n^2}{2}f''(c)+O(e_n^3), \] \[ f(c+e_{n-1})= e_{n-1} f'(c)+\frac{e_{n-1}^2}{2}f''(c)+O(e_{n-1}^3). \] Subtract: \[ f(c+e_n)-f(c+e_{n-1}) =(e_n-e_{n-1})f'(c)+\frac{(e_n^2-e_{n-1}^2)}{2}f''(c)+\cdots \] \[ =(e_n-e_{n-1})\left[f'(c)+\frac{(e_n+e_{n-1})}{2}f''(c)+\cdots\right]. \]

Substitute these into the Secant formula. The factor \((e_n-e_{n-1})\) cancels, and keeping the leading terms gives \[ e_{n+1} = \frac{f''(c)}{2f'(c)}\,e_n e_{n-1} + \text{higher-order terms}. \] Let \[ M_1=\frac{f''(c)}{2f'(c)}, \] so the dominant error relation is \[ e_{n+1}\approx M_1 e_n e_{n-1}. \]

To find the order of convergence, assume \[ e_{n+1}\approx M e_n^{\,p}, \qquad e_n\approx M e_{n-1}^{\,p}. \] From the second relation, \[ e_{n-1}\approx \left(\frac{e_n}{M}\right)^{1/p}. \] Substitute into \(e_{n+1}\approx M_1 e_n e_{n-1}\): \[ e_{n+1}\approx M_1 e_n\left(\frac{e_n}{M}\right)^{1/p} =\frac{M_1}{M^{1/p}}\,e_n^{\,1+\frac{1}{p}}. \] But also \(e_{n+1}\approx M e_n^{\,p}\), so the exponents must match: \[ p = 1+\frac{1}{p}. \] Hence \[ p^2-p-1=0 \quad\Rightarrow\quad p=\frac{1+\sqrt{5}}{2}\approx 1.618. \]

Therefore, the Secant method converges with order \[ p=\frac{1+\sqrt{5}}{2}\approx 1.618. \]

From the dominant error relation for the Secant method,

\[ e_{n+1} \approx M_1 e_n e_{n-1}, \qquad M_1 = \frac{f''(c)}{2f'(c)}, \]

assume the asymptotic form

\[ e_{n+1} \approx C e_n^{\,p}, \]

where \(p = \dfrac{1+\sqrt{5}}{2}\) is the order of convergence. Using the relation \(e_n \approx C e_{n-1}^{\,p}\), we obtain

\[ e_{n-1} \approx \left(\frac{e_n}{C}\right)^{1/p}. \]

Substituting into \(e_{n+1} \approx M_1 e_n e_{n-1}\),

\[ e_{n+1} \approx M_1 e_n \left(\frac{e_n}{C}\right)^{1/p} = \frac{M_1}{C^{1/p}}\, e_n^{\,1+\frac{1}{p}}. \]

Since \(1+\frac{1}{p}=p\), comparison with \(e_{n+1}\approx C e_n^{\,p}\) gives

\[ C = \frac{M_1}{C^{1/p}}. \]

Hence

\[ C^{\,1+\frac{1}{p}} = M_1. \]

Because \(1+\frac{1}{p}=p\), we obtain

\[ C^{\,p} = M_1, \qquad C = |M_1|^{\,1/p}. \]

Using \(M_1=\dfrac{f''(c)}{2f'(c)}\) and \(\dfrac{1}{p}=\dfrac{\sqrt{5}-1}{2}\),

\[ C = \left| \frac{f''(c)}{2f'(c)} \right|^{\frac{\sqrt{5}-1}{2}}. \]

Therefore, for the Secant method,

\[ |x_{n+1}-c| \approx \left| \frac{f''(c)}{2f'(c)} \right|^{\frac{\sqrt{5}-1}{2}} |x_n-c|^{\frac{\sqrt{5}+1}{2}}. \]

Thus, the Secant method converges superlinearly with order \(p=\dfrac{1+\sqrt{5}}{2}\).

Fixed Point Method

Let \(c\) be a fixed point.

\[ g(c) = c, \quad f(c) = 0 \tag{fpm-1} \] \[ x_{n+1} = g(x_n) \]

Consider,

\[ \begin{aligned} e_{n+1} &= x_{n+1} - c \\ &= g(x_n) - g(c) \\ &= g(c + e_n) - g(c) \\ &= g(c) + \frac{e_n}{1!}g'(c) + \frac{e_n^2}{2!}g''(c) + \ldots \end{aligned} \] \[ e_{n+1} = \frac{e_n}{1!}g'(c) + \frac{e_n^2}{2!}g''(c) + \ldots \tag{fpm-2} \]

Case 1

Let \(g(c) = c\), \(g'(c) \neq 0\)

\[ (fpm-2) \implies e_{n+1} \simeq Me_n, \quad M = g'(c), \; p = 1 \]

Case 2

Let \(g(c) = c\), \(g'(c) = 0\), \(g''(c) \neq 0\)

\[ (fpm-2) \implies e_{n+1} \simeq Me_n^2, \quad M = \frac{g''(c)}{2!}, \; p = 2 \]

In general, for order \(p\),

\[ g'(c) = g''(c) = \ldots = g^{(p-1)}(c) = 0, \quad g^p(c) \neq 0 \]

Example

Given

\[ x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}, \quad n = 0,1,2,\ldots \]

and

\[ f(c) = 0, \quad f'(x) \neq 0 \]

Solution:

Let \( x_{n+1} = g(x_n) \), where

\[ g(x) = x - \frac{f(x)}{f'(x)}. \]

1. Show that \( g(c) = c \)

\[ g(c) = c - \frac{f(c)}{f'(c)} = c - \frac{0}{f'(c)} = c. \]

Thus, \( c \) is a fixed point of \( g \).

2. Compute \( g'(x) \)

\[ g(x) = x - \frac{f(x)}{f'(x)}. \] Differentiate using the quotient rule: \[ \left(\frac{f(x)}{f'(x)}\right)' = \frac{f'(x)f'(x) - f(x)f''(x)}{(f'(x))^2} = \frac{(f'(x))^2 - f(x)f''(x)}{(f'(x))^2}. \] Therefore, \[ g'(x) = 1 - \frac{(f'(x))^2 - f(x)f''(x)}{(f'(x))^2}. \] Simplifying: \[ g'(x) = \frac{f(x)f''(x)}{(f'(x))^2}. \] Evaluate at \( x = c \): \[ g'(c) = \frac{f(c)f''(c)}{(f'(c))^2} = \frac{0 \cdot f''(c)}{(f'(c))^2} = 0. \]

3. Compute \( g''(c) \)

Since \[ g'(x) = \frac{f(x)f''(x)}{(f'(x))^2}, \] differentiating and evaluating at \( x = c \) gives \[ g''(c) = \frac{f''(c)}{f'(c)} \neq 0 \] (provided \( f''(c) \neq 0 \)).

4. Conclude quadratic convergence

Using Taylor expansion of \( g(x) \) about \( c \): \[ g(x) = g(c) + g'(c)(x-c) + \frac{g''(c)}{2}(x-c)^2 + O((x-c)^3). \] Since \[ g(c)=c, \quad g'(c)=0, \] we obtain \[ x_{n+1} - c = \frac{g''(c)}{2}(x_n - c)^2 + O((x_n - c)^3). \] Thus, \[ e_{n+1} = K e_n^2 + O(e_n^3), \] where \( e_n = x_n - c \) and \[ K = \frac{f''(c)}{2f'(c)}. \]

Therefore, Newton's method converges quadratically near the root \( c \).
