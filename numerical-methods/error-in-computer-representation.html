Error In Computer Representation

Suppose that \(x^*\) is an approximation to \(x\).

The absolute error is \[ |x - x^*| \] and the relative error is \[ \frac{|x - x^*|}{|x|} \] provided that \( x \ne 0 \).

Example 1

Determine the absolute and relative errors when approximating \(x\) by \(x^*\) when

(a) \(x = 0.3000 \times 10^1 \) and \( x^* = 0.3100 \times 10^1 \);

(b) \(x = 0.3000 \times 10^{-3} \) and \( x^* = 0.3100 \times 10^{-3}\);

(c) \(x = 0.3000 \times 10^4 \) and \( x^* = 0.3100 \times 10^4 \).

Solution

(a) For \(x = 0.3000 \times 10^1\) and \(x^* = 0.3100 \times 10^1\),

Absolute Error

\[ |x - x^*| = |3.000 - 3.100| = 0.1, \]

Relatve Error

\[ \frac{|x - x^*|}{|x|} = \frac{0.1}{3.000} = 0.03333\ldots = 0.3333\overline{3} \times 10^{-1}. \]

(b) For \(x = 0.3000 \times 10^{-3}\) and \(x^* = 0.3100 \times 10^{-3}\),

Absolute Error

\[ |x - x^*| = |0.3000 - 0.3100|\times 10^{-3} = 0.0100 \times 10^{-3} = 0.1 \times 10^{-4}, \]

Relatve Error

\[ \frac{|x - x^*|}{|x|} = \frac{0.1 \times 10^{-4}}{0.3000 \times 10^{-3}} = 0.03333\ldots = 0.3333\overline{3} \times 10^{-1}. \]

(c) For \( p = 0.3000 \times 10^4 \) and \( p^* = 0.3100 \times 10^4 \),

Absolute Error

\[ |x - x^*| = |0.3000 - 0.3100|\times 10^4 = 0.0100 \times 10^4 = 0.1 \times 10^3, \]

Relatve Error

\[ \frac{|x - x^*|}{|x|} = \frac{0.1 \times 10^3}{0.3000 \times 10^4} = 0.03333\ldots = 0.3333\overline{3} \times 10^{-1}. \]

This shows that the same relative error, \( 0.3333\overline{3} \times 10^{-1} \), occurs for widely varying absolute errors. As a measure of accuracy, the absolute error can be misleading, whereas the relative error is often more meaningful because it takes into consideration the size of the value.
