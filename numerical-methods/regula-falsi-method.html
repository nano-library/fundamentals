Regula-Falsi Method

Also known as False Position Method

It is an iterative method that combines elements of the Bisection method and the Secant method.

It includes a test to ensure that the root is always bracketed between successive iterations. Although it is not a method generally recommended, but it illustrates how bracketing can be incorporated.

Suppose \(f\) is a continuous function defined on the interval \([a, b]\), with \(f(a)\) and \(f(b)\) of opposite sign. By the Intermediate Value Theorem, it follows that \(f(x)\) has at least one root in the interval \([a, b]\).

\[ f(a) \cdot f(b) < 0 \]

The method typically converges faster than the Bisection method, especially when the function behaves similarly to a linear function near the root.

To find an approximation for the root of the equation \(f(x) = 0\).

The steps of the Regular-Falsi method are as follows:

1. Initial Points

Choose initial approximations \(x_0\) and \(x_1\) with \(f(x_0) \cdot f(x_1) < 0.

2. Compute the Interpolated Point

The next approximations \(x_n\) are chosen in the same manner as in the Secant method.

\[ x_n = b - \frac{f(b) \cdot (b-a)}{f(b) - f(a)} \]

3. Update the Intervals

- If \(f(x_n) \cdot f(a) < 0\), set the interval to \([a, x_n]\)

- If \(f(x_n) \cdot f(b) < 0\), set the interval to \([x_n, b]\)

4. Iteration

Repeat steps 2 and 3 until the solution coverges to a desired accuracy level.
