Regula-Falsi Method

Also known as False Position Method

It is an iterative method that combines elements of the Bisection method and the Secant method.

It includes a test to ensure that the root is always bracketed between successive iterations. Although it is not a method generally recommended, but it illustrates how bracketing can be incorporated.

Suppose \(f\) is a continuous function defined on the interval \([a, b]\), with \(f(a)\) and \(f(b)\) of opposite sign. By the Intermediate Value Theorem, it follows that \(f(x)\) has at least one root in the interval \([a, b]\).

\[ f(a) \cdot f(b) < 0 \]

The method typically converges faster than the Bisection method, especially when the function behaves similarly to a linear function near the root.

To find an approximation for the root of the equation \(f(x) = 0\).

The steps of the Regular-Falsi method are as follows:

1. Initial Points

Choose initial approximations \(x_0\) and \(x_1\) with \(f(x_0) \cdot f(x_1) < 0\).

2. Compute the Interpolated Point

The next approximation is chosen using the same formula as the Secant method (the line through the two bracket endpoints).

\[ x_{n+1}=x_n - f(x_n)\,\frac{x_n-x_{n-1}}{f(x_n)-f(x_{n-1})}. \]

3. Update the Intervals

- If \(f(x_{n+1}) \cdot f(x_{n-1}) < 0\), set the interval to \([x_{n-1}, x_{n+1}]\)

- If \(f(x_{n+1}) \cdot f(x_n) < 0\), set the interval to \([x_{n+1}, x_n]\)

4. Iteration

Repeat steps 2 and 3 until the solution converges to a desired accuracy level.

Example 1

Find the smallest positive real root of \(\cos(x) - x e^x = 0\), correct to three decimal places.

Solution:

Let \(x_0=0.5,\; x_1=0.6\)

Let \(f(x) = \cos(x) - x e^x\)

\[ f(0.5)=\cos(0.5)-0.5e^{0.5}\approx 0.0532219 \] \[ f(0.6)=\cos(0.6)-0.6e^{0.6}\approx -0.2679357 \]

Hence, a root lies in \([0.5,\,0.6]\).

To obtain the root correct to three decimal places, the error in the approximation must be less than half a unit in the third decimal place.

If a number is correct to three decimal places, then rounding will not change its value provided the error is less than

\[ \frac{1}{2}\times 10^{-3} = 5\times 10^{-4}. \]

Since the exact root is unknown, the absolute error \(|x - \alpha|\) cannot be computed directly. Hence, we use the difference between successive approximations as an estimate of the error. Therefore, we require

\[ |x_{n+1} - x_n| \le 5\times 10^{-4}, \]

which ensures that the approximation is accurate to three decimal places.

The Regular-Falsi (False Position) iterate uses the secant line through the current bracket endpoints.

\[ x_{n+1}=x_n - f(x_n)\,\frac{x_n-x_{n-1}}{f(x_n)-f(x_{n-1})}. \]

Iteration 1

\[ x_2 = x_1 - f(x_1)\,\frac{x_1-x_{0}}{f(x_1)-f(x_{0})} \] \[ x_2 = 0.6 - (-0.2679357)\frac{0.6-0.5}{-0.2679357-0.0532219} \] \[ x_2 = \approx 0.5165719 \] \[ f(x_2)=f(0.5165719)\approx 3.60274\times 10^{-3} > 0 \]

Update the bracket:

\[ f(x_2)\cdot f(x_0) > 0,\qquad f(x_2)\cdot f(x_1) < 0 \]

So the new interval is \([x_2,\,x_1]=[0.5165719,\,0.6]\).

Iteration 2

\[ x_3 = x_2 - f(x_2) \cdot \frac{x_2-x_1}{f(x_2)-f(x_1)} \approx 0.5176788 \] \[ f(x_3)=f(0.5176788)\approx 2.38932\times 10^{-4} > 0 \]

Update the bracket:

\[ f(x_3)\cdot f(x_2) > 0,\qquad f(x_3)\cdot f(x_1) < 0 \]

So the new interval is \([x_3,\,x_1]=[0.5176788,\,0.6]\).

Iteration 3

\[ x_4 = x_3 - f(x_3)\,\frac{x_3-x_1}{f(x_3)-f(x_1)} \approx 0.5177522 \] \[ |x_4-x_3|\approx 7.33\times 10^{-5} < 5\times 10^{-4} \]

Therefore, the smallest positive root correct to three decimal places is

\[ x \approx 0.518. \]
