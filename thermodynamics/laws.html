Laws of Thermodynamics

Zeroth Law of Thermodynamics

If system A is in thermal equilibrium with system C, and system B is also in thermal equilibrium with system C, then A and B are in thermal equilibrium with each other.

First Law of Thermodynamics (Conservation of Energy)

The change in internal energy \(U\) for any transition between two equilibrium states is

\[ \Delta U = Q - W \]

differential form

\[ dU = \delta Q - \delta W. \]

Here \(Q\) is heat added to the system. \(W\) is work done by the system (so if work is done on the system, then \(W<0\)).

State function vs path function

Internal energy \(U\) is a state variable. For a change from state A to state B, different paths (1,2,3,...) can have different \(Q\) and \(W\), but the same \(\Delta U\).

\[ Q_1 - W_1 = Q_2 - W_2 = Q_3 - W_3 = \cdots \] \[ \Delta U_1 = \Delta U_2 = \Delta U_3 = \cdots \]

Problem 1

A system goes from A to B with heat supplied \(Q = 400\,\text{J}\) and work done by the system \(W = 100\,\text{J}\).

(i) Change in internal energy from A to B \[ \Delta U_{A\to B} = Q - W = 400 - 100 = 300\,\text{J}. \] (ii) Change in internal energy from B to A (same endpoints) \[ \Delta U_{B\to A} = -\Delta U_{A\to B} = -300\,\text{J}. \] (iii) From B to A along a different path where \(W' = 400\,\text{J}\) of work is done on the system

“Work done on the system” means work done by the system is negative: \(W = -400\,\text{J}\).

\[ \Delta U_{B\to A} = Q - W \;\Rightarrow\; -300 = Q - (-400) = Q + 400 \] \[ Q = -700\,\text{J}. \]

So the system does not absorb heat; it releases \(700\,\text{J}\) of heat to the surroundings (since \(Q<0\)).

Second Law of Thermodynamics

Free expansion idea (insulated container, vacuum on one side)

If an ideal gas expands into vacuum inside a thermally insulated container, then

\[ W = 0,\qquad Q = 0 \;\Rightarrow\; \Delta U = 0. \]

The first law doesn't forbid the reverse process (gas “un-expanding” back), but nature doesn't spontaneously do that. The second law rules the arrow of spontaneous change.

Clausius Statement

Heat never flows spontaneously from a colder body to a hotter body.

Kelvin-Planck Statement < It is impossible to construct a cyclic device whose only net effect is to extract heat from a single resp> ervoir and convert it completely into work.

Entropy Statement

For an isolated system,

\[ \Delta S \ge 0, \]

with equality for a reversible process and strict inequality for an irreversible process.

Third Law of Thermodynamics

As the temperature approaches absolute zero, the entropy approaches a constant minimum value. For a perfect crystal,

\[ \lim_{T\to 0} S(T) = 0. \]

A common operational consequence: reaching \(T=0\) exactly by a finite number of processes is impossible.
