Fundamental Thermodynamic Identities

1.

\[ dU = TdS - PdV + \mu dN \]

2.

\[ dH = TdS + VdP + \mu dN \]

3.

\[ dF = -SdT - PdV + \mu dN \]

4.

\[ dG = - SdT + VdP + \mu dN \]

First Fundamental Identity (simple compressible system with variable \(N\))

Consider a simple compressible system with variable particle number:

\[ U = U(S,V,N). \]

Taking the total differential:

\[ dU = \left(\frac{\partial U}{\partial S}\right)_{V,N} dS + \left(\frac{\partial U}{\partial V}\right)_{S,N} dV + \left(\frac{\partial U}{\partial N}\right)_{S,V} dN. \]

Define the conjugate (intensive) variables:

\[ T \equiv \left(\frac{\partial U}{\partial S}\right)_{V,N}, \qquad \mu \equiv \left(\frac{\partial U}{\partial N}\right)_{S,V}. \]

For a simple compressible system, the mechanical conjugate of \(V\) is:

\[ -P \equiv \left(\frac{\partial U}{\partial V}\right)_{S,N} \quad \Longrightarrow \quad P = -\left(\frac{\partial U}{\partial V}\right)_{S,N}. \]

Hence:

\[ \boxed{\,dU = T\,dS - P\,dV + \mu\,dN\,}. \]

Second Fundamental Identity (Enthalpy form)

Define enthalpy:

\[ H \equiv U + PV. \]

Differentiating:

\[ dH = dU + P\,dV + V\,dP. \]

Substitute \(dU = T\,dS - P\,dV + \mu\,dN\):

\[ dH = \left(T\,dS - P\,dV + \mu\,dN\right) + P\,dV + V\,dP = T\,dS + V\,dP + \mu\,dN. \]

Therefore:

\[ \boxed{\,dH = T\,dS + V\,dP + \mu\,dN\,}. \]

Since \(H\) naturally depends on \((S,P,N)\):

\[ H = H(S,P,N), \] \[ dH = \left(\frac{\partial H}{\partial S}\right)_{P,N} dS + \left(\frac{\partial H}{\partial P}\right)_{S,N} dP + \left(\frac{\partial H}{\partial N}\right)_{S,P} dN. \]

Comparing with \(dH = T\,dS + V\,dP + \mu\,dN\), we identify:

\[ T = \left(\frac{\partial H}{\partial S}\right)_{P,N}, \qquad V = \left(\frac{\partial H}{\partial P}\right)_{S,N}, \qquad \mu = \left(\frac{\partial H}{\partial N}\right)_{S,P}. \]

Note (common correction): The correct identity is \(V = \left(\frac{\partial H}{\partial P}\right)_{S,N}\), not \( \left(\frac{\partial H}{\partial V}\right)_{S,N}\).

Third Fundamental Identity (Helmholtz free energy form)

Define Helmholtz free energy:

\[ F \equiv U - TS. \]

Differentiate:

\[ dF = dU - T\,dS - S\,dT. \]

Substitute \(dU = T\,dS - P\,dV + \mu\,dN\):

\[ dF = (T\,dS - P\,dV + \mu\,dN) - T\,dS - S\,dT = -S\,dT - P\,dV + \mu\,dN. \]

Hence:

\[ \boxed{\,dF = -S\,dT - P\,dV + \mu\,dN\,}. \]

Natural variables:

\[ F = F(T,V,N). \]

Total differential:

\[ dF = \left(\frac{\partial F}{\partial T}\right)_{V,N} dT + \left(\frac{\partial F}{\partial V}\right)_{T,N} dV + \left(\frac{\partial F}{\partial N}\right)_{T,V} dN. \]

Comparing with \(dF = -S\,dT - P\,dV + \mu\,dN\):

\[ S = -\left(\frac{\partial F}{\partial T}\right)_{V,N}, \qquad P = -\left(\frac{\partial F}{\partial V}\right)_{T,N}, \qquad \mu = \left(\frac{\partial F}{\partial N}\right)_{T,V}. \]

Fourth Fundamental Identity (Gibbs free energy form)

Define Gibbs free energy:

\[ G \equiv H - TS = U + PV - TS. \]

Differentiate:

\[ dG = dH - T\,dS - S\,dT. \]

Use \(dH = T\,dS + V\,dP + \mu\,dN\):

\[ dG = (T\,dS + V\,dP + \mu\,dN) - T\,dS - S\,dT = -S\,dT + V\,dP + \mu\,dN. \]

Therefore:

\[ \boxed{\,dG = -S\,dT + V\,dP + \mu\,dN\,}. \]

Natural variables:

\[ G = G(T,P,N). \]

Total differential:

\[ dG = \left(\frac{\partial G}{\partial T}\right)_{P,N} dT + \left(\frac{\partial G}{\partial P}\right)_{T,N} dP + \left(\frac{\partial G}{\partial N}\right)_{T,P} dN. \]

Comparing with \(dG = -S\,dT + V\,dP + \mu\,dN\):

\[ S = -\left(\frac{\partial G}{\partial T}\right)_{P,N}, \qquad V = \left(\frac{\partial G}{\partial P}\right)_{T,N}, \qquad \mu = \left(\frac{\partial G}{\partial N}\right)_{T,P}. \]

Extensive and Intensive Variables

Extensive Variables

\(V,\,N,\,S,\,U,\,H,\,F,\,G\)

Intensive Variables

\(T,\,P,\,\mu,\) density (e.g. \(N/V\), \(m/V\))

Notes

  • Multiplying an intensive variable by an extensive variable gives an extensive quantity.
    \(T S\) is extensive, \(P V\) is extensive, \(\mu N\) is extensive.
  • You can add/subtract only “same-type” quantities:
    extensive with extensive (OK), intensive with intensive (OK).
    Example: \(U + PV\) is valid because both are extensive. But \(U + P\) is not meaningful (extensive + intensive).
