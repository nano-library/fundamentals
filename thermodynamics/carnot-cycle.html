Carnot Cycle

Efficiency is defined as

\[ e = \frac{W}{Q_h}. \tag{1} \]

For a cyclic process, the system returns to its initial state, so

\[ \Delta U_{\text{cycle}} = 0. \]

From the First Law, \(\Delta U = Q - W\), hence for the cycle

\[ 0 = Q_{\text{net}} - W \quad \Rightarrow \quad W = Q_{\text{net}}. \]

If the engine absorbs heat \(Q_h\) from the hot reservoir and rejects \(Q_c\) to the cold reservoir, then

\[ W = Q_h - Q_c. \]

So efficiency becomes

\[ e = \frac{Q_h - Q_c}{Q_h} = 1 - \frac{Q_c}{Q_h}. \tag{2} \]

Carnot Engine Between \(T_h\) and \(T_c\)

For a reversible Carnot cycle, the heat transfers occur isothermally: \(Q_h\) at \(T_h\) and \(Q_c\) at \(T_c\). For a reversible cycle,

\[ \frac{Q_c}{Q_h} = \frac{T_c}{T_h}. \]

Substitute this into (2):

\[ e = 1 - \frac{T_c}{T_h}. \]

This is the maximum possible efficiency for any engine operating between these two temperatures.

Problem (Carnot Engine)

A Carnot engine operates between \(T_c = 300\,\text{K}\) and \(T_h\). If \(e = 60\% = 0.60\), find \(T_h\). If the engine delivers \(W = 300\,\text{J}\) per cycle, find \(Q_h\) and \(Q_c\).

(i) Find \(T_h\)
\[ e = 1 - \frac{T_c}{T_h} \;\Rightarrow\; 0.60 = 1 - \frac{300}{T_h} \] \[ \frac{300}{T_h} = 0.40 \;\Rightarrow\; T_h = \frac{300}{0.40} = 750\,\text{K}. \]

\(\boxed{T_h = 750\,\text{K}}\)

(ii) Find heat absorbed \(Q_h\)
\[ e = \frac{W}{Q_h} \;\Rightarrow\; Q_h = \frac{W}{e} = \frac{300}{0.60} = 500\,\text{J}. \]

\(\boxed{Q_h = 500\,\text{J}}\)

(iii) Find heat rejected \(Q_c\)
\[ W = Q_h - Q_c \;\Rightarrow\; Q_c = Q_h - W = 500 - 300 = 200\,\text{J}. \]

\(\boxed{Q_c = 200\,\text{J}}\)

Quick Consistency Check
\[ \frac{Q_c}{Q_h} = \frac{200}{500} = 0.40 \quad\text{and}\quad \frac{T_c}{T_h} = \frac{300}{750} = 0.40 \]

Matches Carnot relation, so the numbers are self-consistent.
